# AirBear PWA Project Brief

## Project Overview
**AirBear** is a revolutionary solar-powered ride-sharing platform that combines eco-conscious transportation with onboard mobile bodegas in Binghamton, NY. The platform features a triple-role system (Admin, Driver, User) and provides a complete PWA experience with real-time mapping, inventory management, and sustainable transportation solutions.

## Core Mission
*"AirBear flair, ride without a care—solar power in the air!"*

Provide zero-emission transportation while supporting local businesses and reducing carbon emissions through solar-powered vehicles with integrated shopping experiences.

## Strategic Goals
1. **Sustainability**: 100% solar-powered fleet with real-time environmental impact tracking
2. **Ecosystem**: Support local businesses through mobile bodega partnerships
3. **User Experience**: Seamless PWA experience across all devices with offline support
4. **Revenue Diversification**: Multiple monetization streams (rides, commissions, advertising, merchandise)

## Unique Value Proposition
- Exclusive CEO t-shirt promo providing daily free rides ($100+ value)
- Interactive Binghamton map with 16 strategic real-world locations
- AI-powered ride matching and eco-optimization
- Progressive Web App with native app installation capabilities

## Target Market
- **Primary**: Eco-conscious commuters in Binghamton, NY
- **Secondary**: Local businesses seeking mobile retail opportunities
- **Tertiary**: Corporations interested in green transportation solutions

## Success Metrics
- Ride bookings and completion rates
- User retention and engagement
- CO₂ emissions reduction tracking
- Mobile bodega revenue for vendors
- PWA installation and offline usage rates

## Technical Scope
- Full-stack TypeScript application (React frontend, Node.js backend)
- Real-time mapping and geolocation services
- Payment processing via Stripe
- PostgreSQL database with relationships
- PWA capabilities with service workers and caching
- Responsive design with advanced animations

## Business Model
- Direct ride booking fees
- Commission on mobile bodega sales
- Exclusive merchandise sales
- Tiered advertising system (Silver/Gold/Platinum)
